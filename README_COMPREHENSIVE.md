## рҹҡҖ Overview

**SynQ** is a revolutionary programming language and platform that combines the simplicity of Python, the power of C/C++, the security of Rust, and quantum computing capabilities. SynQ represents 15 phases of development, 60,000+ lines of code, and a vision to create the most powerful, secure, and versatile programming language ecosystem.

### Core Vision

> **Build a universal programming language that is:**
> - **Better than Python** in every way (performance, safety, features)
> - **More powerful than Java** in every way (speed, security, flexibility)
> - **Fastest to compile** with 100x faster incremental builds
> - **Most secure** with military-grade security
> - **Most compatible** with 27 programming languages
> - **Most comprehensive** with 500+ standard library functions
> - **Most user-friendly** with professional IDE and tools

## рҹҡҖ Getting Started

### Installation

```bash
# macOS
brew install synq

# Ubuntu/Debian
curl -fsSL https://synq.dev/install.sh | sudo bash

# Windows
choco install synq

# Docker
docker pull synqsplicer/synq:latest
```

### Create Your First Project

```bash
synq new my_project
cd my_project
synq run
```

### Launch IDE

```bash
synq ide
```

## рҹ”§ Development

### Building from Source

```bash
git clone https://github.com/TangoSplicer/SynQ.git
cd SynQ

# Build compiler
cd compiler
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build

# Build backend
cd ../backend
pip install -r requirements.txt

# Build frontend
cd ../frontend
npm install
npm run build
```

### Running Tests

```bash
# Compiler tests
cd compiler/build
ctest

# Backend tests
cd backend
pytest tests/

# Frontend tests
cd frontend
npm test
```

### Running CI/CD Locally

```bash
# Using act (GitHub Actions locally)
act push -j compiler-ci
act push -j backend-ci
act push -j frontend-ci
```
